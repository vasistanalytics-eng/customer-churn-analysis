# Customer Churn Insights & Retention Dashboard

## Objective
Analyze customer churn behavior and identify key drivers of attrition in a subscription-based telecom business.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI

## Key Insights
- Month-to-month customers have the highest churn rate (~43%)
- New customers (0–12 months) show highest churn (~48%)
- Electronic check users have highest churn (~45%)
- Long-term contracts significantly reduce churn
